# 🚨 MÉMOIRE LUKAPA - Système d'Alarme Connecté IoT  ## 📌 Description du Projet  Ce projet est un **système d'alarme intelligent** réalisé dans le cadre d'un travail de fin d'études. Il utilise un **ESP32**, un **capteur ultrason HC-SR04** et une **sirène** pour détecter les intrusions et alerter à distance via une interface web.
